# 3: 设置IP

# 3.1 分配随机IP

分配静态IP请看3.2

service network restart

ip addr

# 3:永久关闭Selinux

vim /etc/sysconfig/selinux

SELINUX=enforcing 改为 SELINUX=disabled 重启服务reboot
